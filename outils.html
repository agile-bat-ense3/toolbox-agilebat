Les outils de la toolbox

Les outils référencés ici ont été sélectionnés selon trois critères : pertinence pour les questions de phase amont, accessibilité pour les étudiants (gratuit ou licence école), et complémentarité entre eux. Si tu cherches plutôt par question, reviens sur la page d'accueil.

Chaque outil est noté selon des critères pratiques : accessibilité (gratuit/payant), compatibilité OS (Mac/Windows/web), et niveau de prise en main (facile / intermédiaire / avancé).

Outils Esquisse

  • Autodesk Forma Analyse de site, ensoleillement, masques, vents Web · Gratuit étudiants · Facile
  • La Maison Lambda Bilan énergétique simplifié, comparaison de variantes Web · Gratuit · Facile
  • Sunearthtools Course du soleil, diagramme solaire Web · Gratuit · Immédiat
  • Climate Consultant Analyse climatique et stratégies passives Mac/Windows · Gratuit · Facile
  • Ubakus Calcul de U et comportement hygrothermique de paroi Web · Gratuit · Facile
  • Vizcab Explo / Campus Premier ordre de grandeur carbone Web · Gratuit (Campus) · Intermédiaire
  • Mi@ep Compacité du bâtiment Web · Gratuit · Immédiat

Outils APS

  • Pleiades Simulation thermique dynamique, calculs RE2020 Windows · Payant (licence école) · Intermédiaire
  • OpenStudio STD via interface graphique sur moteur EnergyPlus Mac/Windows · Gratuit · Intermédiaire
  • EnergyPlus Moteur de simulation thermique de référence Mac/Windows · Gratuit · Avancé
  • Rhino + Grasshopper Analyse paramétrique, comparaison massive de variantes (Ladybug/Honeybee) Mac/Windows · Rhino payant · Avancé

Comment cette toolbox a été construite

Quatre outils ont été testés en profondeur sur un cas d'étude réel (la maison en terre PhITEM à Saint-Martin-d'Hères) : Pleiades, OpenStudio, EnergyPlus et Rhino+Grasshopper. Les autres sont documentés à partir de la documentation officielle et de retours d'usage par des architectes en exercice.
